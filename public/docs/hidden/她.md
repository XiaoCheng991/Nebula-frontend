# 她

> 2026/06/10 Wednesday

> - 不爱吃金针菇、胡萝卜、几乎所有菇类都不爱

她说： “我还没准备好，接受新的一段感情，不是没出来，很难讲，在我完全忘掉之前，草率开始，对人家也不负责，就像是拿人家过渡一样，我要先自己平衡，过得很幸福、很满足，才需要另一个人加入，变得更幸福！”

她说：“我岁数大了，我经不起这些了，我就想要一段安安稳稳的，平平淡淡的感情。”

她说：“我前男友是个All in的人，他一开始追我的时候，真的也是默默做了很多，其实他并不符合我对未来男朋友的期望，但是他太真诚了，让我觉得试试也没什么，然后他一开始就很投入，投入很多。我属于在观望，后来两个人也吵架，我脾气就暴躁，有问题说不明白，我就是解决人，然后他就一直很耐心。”

她说：“我一遍一遍预设，我和瑶子搬家，这么大的事，本来我的东西就多，我俩从宿舍，七八个大箱子，他问了一句，我说我和瑶子找好司机，帮我俩搬到浦口了，他就觉得不需要他了，他直接回家，我不说，我想看他会不会问我，我搬东西，我也生气他没来，他给我发消息，打电话我都没接，然后跟我吵，问我为什么不直接说，说我以为你都计划好了。我也不想主动说，你帮我，他问了我就会说。”

她说：“想找一个 能为我安排的，不用我说，就能想到的。”
